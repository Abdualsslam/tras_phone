# Backend (NestJS)

هذا المجلد مخصص لمشروع الـ Backend باستخدام NestJS.

## البدء

سيقوم المطور بإنشاء مشروع NestJS هنا باستخدام:

```bash
cd backend
npx @nestjs/cli new . --skip-git
```

## الهيكل المتوقع

```
backend/
├── src/
│   ├── app.module.ts
│   ├── main.ts
│   └── ...

└── ...
```
